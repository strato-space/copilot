# Quickstart — Finance Ops Console (MVP)

## Цель
Быстро проверить ключевые сценарии: план‑факт, редактирование факта, lock месяца, прогноз и аналитика.

## Предусловия
- Доступен backend API и фронтенд.
- Загружены тестовые данные (клиенты/проекты/часы/ставки/FX).

## Быстрые проверки (ручные)

1) **Plan‑Fact Grid**
- Открыть «План‑факт»
- Убедиться, что KPI и таблица заполнены на год, фокус 3 месяца
- Изменить «главный месяц» → KPI пересчитываются

2) **Редактирование факта (T&M)**
- Открыть Drawer по проекту/месяцу
- Ввести `billed_hours`
- Проверить пересчёт суммы и запись в audit‑лог

3) **Редактирование факта (Fix, USD + ручной FX)**
- Ввести сумму в USD
- Ввести ручной FX без комментария → ошибка
- Добавить комментарий → сохранение проходит

4) **Закрытие месяца**
- Нажать «Закрыть месяц»
- Убедиться, что правки факта блокируются

5) **Forecast версия + Copy Forecast**
- Создать новую версию из активной
- Выполнить Copy Forecast (fill_zero)
- Убедиться, что пустые ячейки заполнены

6) **Аналитика и alerts**
- Открыть «Аналитика»
- Проверить KPI и список внимания
- Клик по alert переводит к проекту/месяцу

## Smoke checks (API)

- `GET /api/plan-fact?year=2026&focus_month=2026-01&forecast_version_id=...`
- `GET /api/project/{project_id}/month/2026-01`
- `PUT /api/fact/{project_id}/2026-01`
- `POST /api/period/close`
- `POST /api/forecast/copy`
